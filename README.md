## 覆盖服务

业主在小程序首页悬浮 AI 助手发起咨询后，全部办理流程在对话窗内完成、不跳页：

| 服务 | 关键流程 |
|---|---|
| 物业缴费 | 账单 → 选支付方式 → 凭证 |
| 预缴管理费 | 选月数（≥12月9.5折）→ 支付 → 抵扣周期凭证 |
| 报修服务 | 类型/描述/照片/时间 → 工单 → 时间轴追踪 |
| 访客邀请 | 信息 → QR 通行码 → 短信发送 |
| 装修申请 | 类型/工期/图纸 → 审核 → 押金 |
| 放行条申请 | 物品/照片 → 审核 → 电子放行码 |
| 车位管理 | 列表 → 绑车牌 / 续费 / 月租 |
